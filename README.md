# 🚀 AI Supply Chain Dashboard

## 📌 Overview

AI Supply Chain Dashboard is a full-stack Django-based analytics platform designed to help businesses monitor, analyze, and optimize supply chain operations. The system provides interactive dashboards, KPI tracking, anomaly detection, low-stock alerts, supplier performance analysis, and AI-powered recommendations.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/shalinisenthilkumar2021-cmyk/AI_Supply_Chain_Dashboard.git
cd AI_Supply_Chain_Dashboard
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Server

```bash
python manage.py runserver
```
